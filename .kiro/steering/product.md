# TaskVista プロダクト概要

## プロジェクト目標
TaskVista は、チームのタスクとプロジェクト進捗を管理する Web アプリである。Google Sheets による開発・レビュータスク管理を置き換え、作業効率の向上、担当の明確化、モダンな Kanban + Table の二系統ビューを提供する。

## ターゲットユーザー
- 小規模開発チーム（5〜15 人）
- ソフトウェアエンジニア、プロジェクト責任者、技術レビュー参加者（例：中本さんレビューフロー）

## コアバリュー
- ワンクリックで Project を作成し、Sheets を毎回手コピーする必要をなくす
- 明確な Assignee・ステータス遷移・コメントスレッド・添付管理
- Kanban ドラッグ + 詳細テーブルビュー（「場所」「要望」「ステータス」「コメント」などのフィールド）
- 将来の AI 支援（タスク要約、Assignee 推奨）を見据える
- Google Cloud（Navino 環境）への将来移行を想定したアーキテクチャ

## 主要機能の優先順位
1. Project と Task の管理（CRUD）
2. Kanban ボード + Table リスト
3. Task 詳細（コメント + 添付）
4. Dashboard とフィルター
