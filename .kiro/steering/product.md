# TaskVista 产品概述

## 项目目标
TaskVista 是一个团队任务与项目进度追踪 Web 应用，旨在替代 Google Sheets 管理开发/评审任务，提升工作效率、明确任务分配，并提供现代化的 Kanban + Table 双视图。

## 目标用户
- 小型开发团队（5-15 人）
- 软件工程师、项目负责人、技术评审参与者（如中本さんレビュー流程）

## 核心价值
- 一键创建 Project，避免每次手动复制 Sheets
- 清晰的 Assignee + 状态流转 + 评论线程 + 附件管理
- 支持 Kanban 拖拽 + 详细表格视图（支持「場所」「要望」「ステータス」「コメント」等字段）
- 未来支持 AI 辅助（任务总结、Assignee 推荐）
- 架构设计支持未来迁移到 Google Cloud（Navino 环境）

## 关键功能优先级
1. Project 与 Task 管理（CRUD）
2. Kanban 看板 + Table 列表
3. Task 详情（评论 + 附件）
4. Dashboard 与筛选
