# TaskVista 日本語要件（必ず厳守）

## コードコメント
- すべてのコードコメント（//、/* */、#）は**自然な日本語**で記述する
- 例：
  // 案件管理機能
  /* タスクのステータスを更新する */

## UI テキスト
- ユーザー向けの文言（ボタン、タイトル、ラベル、ヒントなど）はすべて**自然な日本語**とする
- 例：
  - 「新規プロジェクト作成」ではなく「新しいプロジェクトを作成」
  - 「保存」ではなく「保存する」
  - ステータス：未着手、進行中、レビュー待ち、完了、保留

## その他
- 変数名・関数名は英語（camelCase）のままとする
- 日本語は正しい敬語・業務用語（日系企業向けの表現）を使う
- エラーメッセージ・成功メッセージも日本語とする
