# TaskVista 日语要求（必须严格遵守）

## 代码注释
- 所有代码注释（//、/* */、#）必须使用**自然日语**
- 示例：
  // 案件管理機能
  /* タスクのステータスを更新する */

## UI 文字
- 所有用户界面文字（按钮、标题、标签、提示信息等）必须使用**自然日语**
- 示例：
  - 「新規プロジェクト作成」ではなく「新しいプロジェクトを作成」
  - 「保存」ではなく「保存する」
  - 状态：未対応、対応中、レビュー待ち、完了

## 其他
- 变量名、函数名继续使用英文（camelCase）
- 日语文本使用正确的敬语/业务用语（日企风格）
- 错误消息、成功提示也使用日语
